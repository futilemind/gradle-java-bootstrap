This is a gradle bootstrap project. 

Useful when you need a quick working setup to test something out or as a project starting point.

Java 1.7 sourceCompatibility and targetCompatibility

Gradle wrapper 1.12

## Plugins:
* jacoco
* java
* eclipse
* idea

## Testing:
* Mockito 1.9.0
* Junit 4+ 

## Build:
./gradlew clean build
